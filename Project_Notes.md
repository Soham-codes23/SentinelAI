# SentinelAI

## Problem Statement
Phishing and malicious URLs are increasing rapidly due to AI-generated scams, fake domains, and social engineering attacks.

## Objective
Build an AI-assisted threat intelligence system capable of detecting suspicious URLs and providing explainable threat analysis.

## Core Features
- URL analysis
- Safe/Malicious prediction
- Threat explanation
- Threat dashboard

## Tech Stack
- Python
- Flask
- sklearn
- HTML/CSS
- Kaggle datasets

## Future Scope
- Browser extension
- Email phishing detection
- Real-time monitoring
- AI chatbot integration
